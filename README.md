# DeployMDFile

## 简介
一款 Obsidian 插件，可以将当前编辑的文件以 Issues 的方式发布到某个仓库


## 使用方式

- 将项目 `clone` 到 **Obsidian** 的插件目录
```bash
git clone git@github.com:DivinerSun/DeployMDFile.git
# git clone https://github.com/DivinerSun/DeployMDFile.git
```

- 安装项目依赖
```bash
yarn
```

- 手动打包安装
```bash
yarn build
```

- 重启 Obsidian
